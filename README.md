# shubhamgoel27.github.io

Personal website built with [Astro](https://astro.build/), based on the [BryanHogan](https://github.com/BryanHogan/bryanhogan) template.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployed automatically to GitHub Pages via GitHub Actions on push to `main`.
